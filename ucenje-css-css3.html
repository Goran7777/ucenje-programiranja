Vodič za učenje CSS3 opisnog jezika

Šta je CSS 3

Kao što smo već pomenuli na glavnoj stranici ovog sajta CSS3 ili na engleskom Cascading Stylesheet je opisni jezik koji služi za ukrašavanje(čitaj dizajn) naših HTML5 stranica i tagova u njima.

Kako CSS3 radi,pa css prvenstveno ima stotine(možda i hiljade) css propertija za razne stilizacije i radi tako što stilizaciju primenjujemo preko imena taga u našem css3 fajlu koji uključujemo u naš HTML fajl ili preko raznih drugih selektora kojima pogadjamo naše DOM elemente u HTML-u.

Selektora ima puno i vremenom ćete širiti vaša znanja o njima kako pogoditi baš svaki element na našoj stranici i primeniti neku stilizaciju na taj element.

Najrasprostranjeniji način primene css-a na html element je na način da damo odredjeni id ili klasu html tagu koji zelimo stilizovati ili ga možemo pogoditi po imenu taga,a ovo poslednje je najbolji način za stilizaciju recimo h1 i h2 html tagova koji se stalno ponavljaju kroz naš websajt a pri tom svi imaju isti dizajn.

CSS3 selektore možete pogledati na W3 CSS3 seletors a sve takozvane CSS3 propertije(čitavu listu) možete pogledati na W3 CSS3 properties i videti u toj dokumentaciji kako i šta koji od njih radi,mada vam to ne preporučujem jer ih zaista ima mnogo,jednostavno je bolji put "guglati" odredjeni property kada vam zatreba.

CSS3 se najbolje uči kroz samo vežbanje pravljenja layouta(skeletona) naših web stranica.

Gde i kako učiti CSS CSS3?

Dakle pored w3 schools dokumentacije kao sto sam pomenuo postoji i MDN(Mozilla Developers Network) što se suvoparne dokumentacije za opisni jezik tiče.

Playlistu možete pronaći na Besplatno učenje CSS3 i ona sadrži 10 video-a od toga "šta je css3?" pa do objašnjenja kako uključiti css u naš html fajl.

Takodje imate tamo i primere sva 3 načina pravljenja skeleta naših web stranica sa kodovima svakog od tri načina a danas postoje 3 osnovna načina postavljanja layout-a(skeleta) a to su "float","flexbox" i najnoviji "grid sistem" za postavljanje layout-a u našim stranicama.

Preporučujem vam ako ste početnik da pogledate tih 10 video-a obavezno jer tamo imate jasne instrukcije u vezi ovog opisnog jezika.
